Nuvora AI Job Assistant
This is a full online AI-powered job assistant built with Streamlit and Google Gemini API.

Features
Resume upload and analysis
Project extraction
AI career chatbot
Dashboard with matched/missing skills
Setup
Clone the repository.
Install dependencies: pip install -r requirements.txt
Add your Gemini API key in .env file:
GEMINI_API_KEY=YOUR_KEY_HERE
Run the app: streamlit run app.py
